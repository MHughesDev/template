# incident rollback

## Purpose
Defines the standard operating sequence for **incident rollback**.

## Minimum workflow
1. Confirm scope and acceptance criteria.
2. Apply change in small increments.
3. Run relevant `make` validation targets.
4. Record evidence in PR and handoff.
