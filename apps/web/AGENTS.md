# apps/web — Frontend AGENTS scope

This is the React 19 frontend, based on Tiangolo's full-stack FastAPI
template. Stack:

- **React 19** + **TypeScript**
- **Vite 7** (with `@vitejs/plugin-react-swc`)
- **TanStack Router** (file-based) and **TanStack Query** for server state
- **Tailwind v4** + **shadcn/ui** + **Radix primitives**
- **react-hook-form** + **zod** for forms
- **biome** for lint/format
- **Playwright** for end-to-end tests
- **openapi-ts** to generate a typed client from the backend's OpenAPI

> Root `AGENTS.md` (two levels up) is authoritative. This file scopes the
> frontend conventions only.

## Conventions

- **Generated code is read-only.** Files under `src/client/` and
  `src/routeTree.gen.ts` are regenerated. To change API shape, change the
  backend and rerun `make generate-client`.
- **Routes:** create files under `src/routes/...`; the TanStack Router
  plugin will regenerate `routeTree.gen.ts` on dev / build.
- **Server state** lives in TanStack Query hooks; **local UI state** lives
  in components. Do not cache server data in `useState`.
- **Forms:** `react-hook-form` + `zod` schema; reuse zod schemas where
  possible (some come from generated `schemas.gen.ts`).
- **Styling:** Tailwind v4 utility classes; shared UI primitives live in
  `src/components/ui/` (shadcn). Prefer composing primitives over creating
  one-off components.
- **Linting and formatting** are owned by Biome. Run `make lint-web`
  (writes safe fixes).
- **Tests** are Playwright e2e under `tests/`. Run `make test-web` (needs
  the stack running, see Playwright docs).

## Common commands

| Task                                 | Command                |
| ------------------------------------ | ---------------------- |
| Install deps                         | `make web-install`     |
| Run dev server (Vite)                | `make dev-web`         |
| Build production bundle              | `cd apps/web && bun run build` |
| Lint (biome)                         | `make lint-web`        |
| Run Playwright tests                 | `make test-web`        |
| Regenerate client from backend       | `make generate-client` |
